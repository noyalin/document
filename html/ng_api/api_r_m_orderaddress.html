OrderAddress API Model

API model information

Description

The address of an order on the marketplace

Properties

Property Required Data Type Description Example Values
AddressLine1 False string The first line of the address
  • 123 Main St.
  • (etc.)
AddressLine2 False string The second line of the address
  • Apt. #137
  • C/O Addison Phelps
  • (etc.)
City False string The city of the address
  • Seattle
  • (etc.)
CompanyName False string The company name of the address
  • Microsoft
  • (etc.)
Country False string The ISO 3166-1 alpha-2 country code of the address
  • US
  • DE
  • MX
  • (etc.)
DaytimePhone False string The daytime phone number of the address
  • 919-123-4567
  • (etc.)
EmailAddress False string The email of the person associated with this address
  • jsmith@example.com
  • (etc.)
EveningPhone False string The evening phone number of the address
  • 919-123-4568
  • (etc.)
FirstName False string The first name of the person associated with this address
  • John
  • (etc.)
LastName False string The last name of the person associated with this address
  • Smith
  • (etc.)
NameSuffix False string The suffix of the person associated with this address
  • Jr.
  • Sr.
  • (etc.)
PostalCode False string The postal code of the address
  • 27560
  • 85204-12345
  • (etc.)
StateOrProvince False string The ISO 3166-2 state/province code of the address
  • NC
  • AK
  • DC
  • (etc.)
