OrderStatus API Model

API model information

Description

The order status of an order on the marketplace

Allowed Values

Both integer and string value representations are acceptable as responses for this model, however, for readability purposes, only the string representation will be sent in requests from Voyageone.

Integer Value String Value
1 Pending
2 ReleasedForShipment
3 AcknowledgedBySeller
4 PartiallyShipped
5 Shipped
6 Canceled
