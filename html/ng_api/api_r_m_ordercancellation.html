OrderCancellation API Model

API model information

Description

The cancellation or refund of an order on the marketplace

Properties

Property Required Data Type Description
Items True OrderItemCancellation array The items to be refunded or canceled
OrderID True string The unique identifier (generated by the site) that represents this order
