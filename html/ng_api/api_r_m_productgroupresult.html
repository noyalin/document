ProductGroupResult API Model

API model information

Description

A result from the processing of a product group

Properties

Property Required Data Type Description
BuyableProductResults True BuyableProductResult array The results of processing each buyable product
SellerSKU True string The unique seller SKU for the product group
Errors False Error array Errors or validation failures that prevent successful processing. Required if not successful
HasErrors False boolean
