Acknowledge order API Endpoint

API endpoint information

URL: https://{your.domain.com}/orders/{id}/acknowledge

Method: POST

Parameter Required Description
id True The ID of the order to acknowledge

Asynchronous response support

Voyageone currently supports only synchronous responses from this endpoint.

Description

Marks the specified order as acknowledged. If there is no order with that ID, return Error ID 6000 (OrderNotFound)

Request Format

This endpoint does not accept a request body.

Response Format

This endpoint returns a/n EmptyObject as its response body (example below):

EmptyObject
