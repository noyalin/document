BuyableProduct API Model

API model information

Description

An item for purchase on the marketplace

Properties

Property Required Data Type Description Example Values
ListingStatus True ListingStatus The listing status of the buyable product
Quantity True integer Quantity available for purchase
  • 0
  • 10
  • 9999999
  • (etc.)
SellerSKU True string A unique seller SKU for the product
  • 123456
  • H1% - L!?@
  • (etc.)
Fields False Field array Fields related to the buyable product
