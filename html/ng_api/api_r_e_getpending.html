Get pending batch API Endpoint

API endpoint information

URL: https://{your.domain.com}/(any)

Method: GET

This endpoint does not accept any URL parameters.

Asynchronous response support

Voyageone currently supports both synchronous and asynchronous responses from this endpoint.

Description

The endpoint to poll for pending batches

Request Format

This endpoint does not accept a request body.

Response Format

This endpoint returns a/n Object as its response body (example below):

  1. {
  2. "ResponseBody": { },
  3. "Status": "Complete",
  4. "PendingUri": null,
  5. "Errors": [ ]
  6. }
