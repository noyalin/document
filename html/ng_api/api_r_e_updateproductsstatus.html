Update products status API Endpoint

API endpoint information

URL: https://{your.domain.com}/products/status

Method: POST

This endpoint does not accept any URL parameters.

Asynchronous response support

Voyageone currently supports both synchronous and asynchronous responses from this endpoint.

Description

Updates the status of all products in the request body and returns a status for each change. Note that in this endpoint, each buyable product is sent as its own product group, with the buyable product's SKU being sent at both the group and the buyable level
