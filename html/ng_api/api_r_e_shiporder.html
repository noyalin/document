Ship order API Endpoint

API endpoint information

URL: https://{your.domain.com}/orders/{id}/ship

Method: POST

Parameter Required Description
id True The ID of the order to ship

Asynchronous response support

Voyageone currently supports only synchronous responses from this endpoint.

Description

Marks the specified order as shipped. If there is no order with that ID, return Error ID 6000 (OrderNotFound)

Request Format

This endpoint accepts a/n ShipRequest as its request body (example below):

  1. {
  2. "ShippedDateUtc": "2017-05-27T05:34:44.4322802Z",
  3. "TrackingNumber": "Z123456789J",
  4. "ShippingCarrier": "Bantha Union",
  5. "ShippingClass": "Express",
  6. "Items": {
  7. "LIGHTSABER_BLUE_MED": 1,
  8. "LIGHTSABER_RED_LG": 4,
  9. "REBEL X-WING": 5
  10. }
  11. }

Response Format

This endpoint returns a/n EmptyObject as its response body (example below):

  1. {
  2. "ResponseBody": { },
  3. "Status": "Complete",
  4. "PendingUri": null,
  5. "Errors": [ ]
  6. }
