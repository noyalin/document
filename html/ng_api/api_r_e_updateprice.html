Update products quantity and price API Endpoint

API endpoint information

URL: https://{your.domain.com}/products/quantityprice

Method: POST

This endpoint does not accept any URL parameters.

Asynchronous response support

Voyageone currently supports both synchronous and asynchronous responses from this endpoint.

Description

Updates the price and quantity of all products in the request body and returns a status for each change
