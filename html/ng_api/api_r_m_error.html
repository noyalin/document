Error API Model

API model information

Description

An error that has occurred on the marketplace API

Properties

Property Required Data Type Description
ID True ErrorID The ErrorID (string or int value acceptable)
ErrorCode False string The error code you want sent to the seller
Message False string The error message you want sent to the seller

Example Model

Use the JSON code below as an example of how this model will be serialized.

  1. {
  2. "ID": "ProductFailedDataValidation",
  3. "ErrorCode": "126",
  4. "Message": "HTML tags are not allowed in the product description field."
  5. }
