ActionResponse API Model

API model information
h3>Description

The container for all responses from the marketplace API

Properties

Property Required Data Type Description
Status True ResponseStatus Indicates to the caller whether to poll for an async response
Errors False Error array Errors or validation failures that prevent successful processing. Required if not successful
HasErrors False boolean
PendingUri False string If Status = Pending, this property is required. Represents the URL to poll for the response. Must be a valid URL value
ResponseBody False nullable object The data of the response. Required if Status is Complete or CompleteWithErrors
