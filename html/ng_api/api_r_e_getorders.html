Get orders API Endpoint

API endpoint information
h3>URL: https://{your.domain.com}/orders?status={status}&limit={limit}

Method: GET

Parameter Required Description
limit False The number of orders to return. Default is no limit
status False The status of orders to return. If no status is specified, orders of any status are returned

Asynchronous response support

Voyageone currently supports both synchronous and asynchronous responses from this endpoint.

Description

Returns a list of all orders matching the filter criteria

Request Format

This endpoint does not accept a request body.

Response Format
