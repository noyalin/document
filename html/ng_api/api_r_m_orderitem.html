OrderItem API Model

API model information

Description

A line item of an order on the marketplace. Corresponds to BuyableProduct

Properties

Property Required Data Type Description Example Values
ID True string A unique identifier for this ordered line item on your site. If your site does not have an identifier, use the same value as the SellerSKU field
  • 123456
  • oB7cJ
  • (etc.)
Quantity True integer The number of units ordered
  • 1
  • 6
  • (etc.)
SellerSku True string The SellerSKU of the ordered BuyableProduct
  • 123456
  • H1% - L!?@
  • (etc.)
UnitPrice True decimal The cost per unit of the ordered line item. Cannot have more than 2 decimal places
  • 12.34
  • 0.78
  • (etc.)

Example Model

Use the JSON code below as an example of how this model will be serialized.

  1. {
  2. "ID": "M000W",
  3. "SellerSku": "REBEL X-WING",
  4. "UnitPrice": 100.00,
  5. "Quantity": 5
  6. }
