ShipRequest API Model

API model information

Description

A request to ship an order on the marketplace

Properties

Property Required Data Type Description Example Values
ShippedDateUtc True UTC date/time The ISO 8601 UTC date/time on which the order was marked as shipped in Voyageone
Items False string dictionary The items in the shipment. Key = Shipped SellerSKU, Value = Shipped Qty
ShippingCarrier False string The carrier company of the shipment
  • FedEx
  • USPS
  • (etc.)
ShippingClass False string The shipping class of the shipment
  • Ground
  • Expedited
  • Freight
  • (etc.)
TrackingNumber False string The tracking number of the shipment
  • 9400115901458007184672
  • (etc.)
