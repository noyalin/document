Get order API Endpoint

API endpoint information

URL: https://{your.domain.com}/orders/{id}

Method: GET

Parameter Required Description
id True The ID of the order to return

Asynchronous response support

Voyageone currently supports only synchronous responses from this endpoint.

Description

Returns the order specified in the URL. If there is no order with that ID, return Error ID 6000 (OrderNotFound)

Request Format

This endpoint does not accept a request body.

Response Format
