Options

Options for configuring your Voyageone integration.
Option Example Description
Base URL https://example.com/api/ The root URL of your VoyageOne OpenAPI
Support Zero Quantity Products True If true, sellers will be permitted to list products that are out of stock (meaning they do not have positive available buyable quantity).
If false, sellers will not be permitted to list out of stock products.
Product Content Update Frequency 6 hours How often Voyageone will attempt to update product data such as title and description.
Quantity and Price Update Frequency 15 minutes How often Voyageone will attempt to update each seller’s quantity and price information.
Order Import Frequency 15 minutes How often Voyageone will attempt to import new seller orders, send shipments, and issue refund and cancellation requests.
API Rate Limits 1000 requests per minute per seller Restricts how many requests can be made on a seller’s behalf to your API in minutes.
Update Batch Size Limits 500 Products / batch
200 Shipments/ batch 		Restricts the maximum number of elements per batch that Voyageone will send to your API
