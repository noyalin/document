Order API Model

API model information

Description

An order on the marketplace

Properties

Property Required Data Type Description Allowed Values Example Values
Currency True string The ISO 4217 code of the currency used for all price amounts on this order
  • USD
  • GBP
  • (etc.)
ID True string The unique identifier (generated by the site) that represents this order
  • 123456
  • oB7cJ
  • (etc.)
Items True OrderItem array The items ordered by the buyer. No two items may have the same SKU or ID
OrderDateUtc True UTC date/time The ISO 8601 UTC date/time indicating when the order was placed
  • 2015-08-11T16:27:28.1529912Z
  • (etc.)
OrderStatus True OrderStatus The current status of the order on the site
RequestedShippingMethod True string The shipping class requested by the buyer
  • Expedited
  • Standard
  • (etc.)
TotalGiftOptionPrice True decimal The total amount paid for the order's gift wrap costs (including taxes if VatInclusive = true). Cannot have more than 2 decimal places
  • 0.00
  • (etc.)
TotalGiftOptionTaxPrice True decimal The total amount paid in taxes for the order's gift wrap costs. Cannot have more than 2 decimal places
  • 0.00
  • (etc.)
TotalPrice True decimal The total amount paid for the order, including taxes and shipping. Cannot have more than 2 decimal places
  • 100.00
  • (etc.)
TotalShippingPrice True decimal The total amount paid for the order's shipping (including taxes if VatInclusive = true). Cannot have more than 2 decimal places
  • 5.00
  • (etc.)
TotalShippingTaxPrice True decimal The total amount paid in taxes for the shipping of the order. Cannot have more than 2 decimal places
  • 0.45
  • (etc.)
TotalTaxPrice True decimal The total amount paid in taxes for the items of the order (does not include shipping tax). Cannot have more than 2 decimal places
  • 22.00
  • (etc.)
VatInclusive True boolean If true, all amounts are understood to include tax
  • true
  • false
BuyerAddress False OrderAddress The billing address associated with the order (the address used during the order's purchase)
DeliverByDateUtc False nullable UTC date/time The ISO 8601 UTC/time date by which the order should be delivered to the shipping address
  • null
  • 2015-08-11T16:27:28.1529912Z
  • (etc.)
OtherFees False decimal The amount charged to the buyer not represented elsewhere in the order (like recycling fee, expedite fee, etc). Cannot have more than 2 decimal places
  • 0.00
  • (etc.)
ShippingAddress False OrderAddress The shipping address associated with the order (the address where the ordered items should be sent)
ShippingLabelURL False string The URL of the shipping label for the order. Must be in valid URL format if provided.
  • http://www.asapsystems.com/instructions/FAQ_FedEx_2D_Barcode_Information.pdf
  • (etc.)
