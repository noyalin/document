Cancel order API Endpoint

API endpoint information

URL: https://{your.domain.com}/orders/{id}/cancel

Method: POST

Parameter Required Description
id True The ID of the order to cancel

Asynchronous response support

Voyageone currently supports only synchronous responses from this endpoint.

Description

Marks the specified items of the specified unshipped order as canceled. If there is no order with that ID, return Error ID 6000 (OrderNotFound)

Request Format

This endpoint accepts a/n OrderCancellation as its request body (example below):

  1. {
  2. "OrderID": "M456W",
  3. "Items": [
  4. {
  5. "ID": "M333W",
  6. "SellerSku": "LIGHTSABER_BLUE_MED",
  7. "Quantity": 1,
  8. "Reason": "BuyerCanceled"
  9. },
  10. {
  11. "ID": "M222W",
  12. "SellerSku": "LIGHTSABER_RED_LG",
  13. "Quantity": 2,
  14. "Reason": "CustomerExchange"
  15. },
  16. {
  17. "ID": "M000W",
  18. "SellerSku": "REBEL X-WING",
  19. "Quantity": 3,
  20. "Reason": "MerchandiseNotReceived"
  21. }
  22. ]
  23. }

Response Format

This endpoint returns a/n EmptyObject as its response body (example below):

  1. {
  2. "ResponseBody": { },
  3. "Status": "Complete",
  4. "PendingUri": null,
  5. "Errors": [ ]
  6. }
