BuyableProductResult API Model

API model information

Description

The result of processing a product by the site

Properties

Property Required Data Type Description
RequestResult True RequestResult Whether the action performed on the Seller SKU succeeded or failed
SellerSKU True string The unique seller SKU for the buyable product
Errors False Error array Errors or validation failures that prevent successful processing. Required if not successful
HasErrors False boolean
MarketPlaceItemID False string The unique ID for this buyable product on the site
URL False string The URL for this buyable product on the site
