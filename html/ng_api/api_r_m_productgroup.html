ProductGroup API Model

API model information

Description

An grouping of one of more items for purchase on the marketplace

Properties

Property Required Data Type Description
BuyableProducts True BuyableProduct array A list of buyable products
SellerSKU True string A unique seller SKU for the product
Fields False Field array Fields related to the product group
