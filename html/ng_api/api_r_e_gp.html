Get products API Endpoint

API endpoint information

URL: https://{your.domain.com}/products?groupFields={groupFields}&buyableFields={buyableFields}

Method: GET

Parameter Required Description
buyableFields False A list of BuyableProduct field names to return. If this query string is not provided or is empty, buyable fields should be returned as an empty collection.
groupFields False A list of ProductGroup field names to return. If this query string is not provided or is empty, group fields should be returned as an empty collection.

Asynchronous response support

Voyageone currently supports both synchronous and asynchronous responses from this endpoint.

Description

Returns a list of all products on the marketplace

Request Format

This endpoint does not accept a request body.

Response Format
