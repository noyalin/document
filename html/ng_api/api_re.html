Requirements

Various requirements that should be implemented in your API integration.
  • Seller SKU needs to be the primary identifier for all product API calls and must be present on order line items.
  • The seller SKU properties must support a length of at least 50 characters and containing all printable ASCII characters, including spaces.
  • You are to supply a list of categories that Voyageone will map to seller products. For an enhanced on-boarding experience, you may provide mappings between your categories and those of another site supported by Voyageone.
